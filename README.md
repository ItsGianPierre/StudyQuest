StudyQuest

Academic trivia game developed in Python for Stanford University's Code in Place program.

Description

StudyQuest is a console-based academic trivia game designed to provide an interactive and educational experience. Players can choose from different categories, answer randomized questions, and receive immediate feedback along with their final score and performance percentage.

Features
Interactive console interface
Player name registration
Multiple trivia categories:
Mathematics
Science

Randomized question order
Immediate feedback after each answer
Final score calculation
